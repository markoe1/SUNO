# Product Hunt Launch Checklist

Prep (2–5 days before)
- [ ] Create PH maker profile(s) and team page
- [ ] Secure a Hunter (optional) or self‑post
- [ ] Add PH badge section to README with repo link
- [ ] Record 60–90s demo (install → fetch 1 clip → post)
- [ ] Prepare 4 gallery images (logo, flow diagram, dashboard, CLI)
- [ ] Write first comment (maker’s story + quick start)
- [ ] DM 25–50 genuine contacts for feedback (no vote begging)

Launch day (Pacific Time)
- [ ] Post at 12:01am PT for full‑day exposure
- [ ] Pin first maker comment with quick start
- [ ] Share on X/LinkedIn/Discord/Reddit subreddits (fit to rules)
- [ ] Answer every comment within 5–10 minutes
- [ ] Ship a small fix or doc improvement during the day

Follow‑up (48 hours)
- [ ] Write “What we learned” post + changelog
- [ ] Thank voters, invite to GitHub Issues/Discord
- [ ] Convert PH traffic to stars/watchers and email capture
