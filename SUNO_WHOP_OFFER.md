## 🎯 What SUNO Is

**Software tool that automates the entire Whop clipping workflow:**

1. Auto-discovers active Whop campaigns
2. Downloads clips from campaigns
3. Posts to TikTok + Instagram Reels + YouTube Shorts (simultaneously)
4. Submits URLs back to Whop automatically
5. Tracks earnings in real-time
6. Auto-retries failed posts

**Result**: Set it up once → runs 24/7 automatically

## 🚀 Onboarding Flow (First 30 Minutes)

### Step 1: Login & Connect (5 min)
1. User signs up on Whop
2. Gets login credentials
3. Logs into SUNO dashboard
4. Clicks "Connect Whop Account"

### Step 2: Whop Authentication (5 min)
1. User logs into Whop in browser (manual — one time only)
2. SUNO saves session (encrypted)
3. Validates connection with test API call
4. Dashboard shows "Whop Connected ✅"

### Step 3: Platform Setup (10 min)
1. User connects TikTok account
2. User connects Instagram account
3. User connects YouTube account
4. SUNO validates all connections

### Step 4: Campaign Setup (5 min)
1. Dashboard auto-discovers active campaigns
2. User selects which campaigns to auto-post
3. User sets posting schedule (optional)
4. Clicks "Start Automation"

### Step 5: Dashboard Orientation (5 min)
1. Show earnings dashboard
2. Explain campaign queue
3. Point to posted clips log
4. Show support contact button

**Total time: ~30 minutes to full automation**
