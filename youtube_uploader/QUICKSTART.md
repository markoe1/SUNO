# YouTube Uploader — Quick Start

## Folder Structure
```
youtube_uploader/
├── credentials.json          ← PLACE YOUR OAUTH CREDENTIALS HERE
├── token.pickle              ← Auto-created after first login
├── upload_video.py           ← Core uploader class
├── simple_upload.py          ← Upload one video (easiest)
├── batch_upload.py           ← Upload multiple videos
├── videos.json.example       ← Example batch config
├── .env.example
├── requirements.txt
├── README.md
└── clips/                    ← Put your video files here
    ├── clip1.mp4
    └── clip2.mp4
```

## Setup (One Time)

### Step 1: Google Cloud OAuth
1. Go to Google Cloud Console
2. Create project "SUNO YouTube"
3. Enable YouTube Data API v3
4. Create OAuth credentials (Desktop application)
5. Download credentials.json

### Step 2: Place Credentials File
Save your downloaded `credentials.json` in this folder:
```
/c/Users/ellio/SUNO-repo/youtube_uploader/credentials.json
```

### Step 3: Install Dependencies
```bash
cd /c/Users/ellio/SUNO-repo/youtube_uploader
pip install -r requirements.txt
```

## Usage

### Option A: Upload One Video (Easiest)

Edit `simple_upload.py`:
```python
video_config = {
    'file_path': 'clips/my_video.mp4',
    'title': 'My SUNO Clip',
    'description': 'Made with SUNO AI',
    'tags': ['suno', 'music'],
    'privacy_status': 'unlisted',
}
```

Run:
```bash
python simple_upload.py
```

**First time**: Browser opens → Click "Authorize" → Done
**Next times**: Uses saved token automatically

### Option B: Upload Many Videos (Batch)

Create `videos.json`:
```json
[
  {
    "file_path": "clips/clip1.mp4",
    "title": "Clip 1",
    "description": "Description",
    "tags": ["suno"],
    "privacy_status": "unlisted",
    "category_id": "10"
  },
  {
    "file_path": "clips/clip2.mp4",
    "title": "Clip 2",
    ...
  }
]
```

Run:
```bash
python batch_upload.py videos.json
```

## Privacy Options
- `'unlisted'` ← Recommended (visible by link, not searchable)
- `'public'` (visible to everyone)
- `'private'` (only you can see)

## Category IDs
- `10` = Music
- `22` = Shorts
- `24` = Comedy
- `28` = Science & Tech

## How OAuth Works

**First run**: Opens browser → You authorize → Token saved locally
**Future runs**: Uses saved token (no browser needed)
**Token expires?**: Auto-refreshes invisibly

## Troubleshooting

| Problem | Solution |
|---------|----------|
| "FileNotFoundError: credentials.json" | Place your Google Cloud OAuth credentials.json in this folder |
| "Invalid client" | Make sure credentials.json is correct and from Google Cloud |
| "Access Denied" | Grant SUNO permission in Google OAuth consent screen |
| "File not found" | Check video file path exists and is correct |

## Next: Integrate with SUNO

Once working, integrate into SUNO orchestrator:
```python
from youtube_uploader.upload_video import YouTubeUploader

uploader = YouTubeUploader()
uploader.authenticate()

for clip in suno_clips:
    uploader.upload_video(
        file_path=clip.path,
        title=clip.title,
        description=f"Generated by SUNO",
        tags=['suno', 'ai-music'],
        privacy_status='unlisted'
    )
```

Done!
